# 智能灌溉 APP

基于 uni-app + HBuilderX 开发的智能灌溉系统手机控制 APP，支持 Android。

## 功能特性

- 蓝牙连接 HC-08 模块
- 实时显示传感器数据（温度、湿度、土壤湿度、光照）
- 历史数据折线图展示
- 远程控制浇水
- 手动/自动扫描连接蓝牙设备
- 通信日志显示

## 数据格式

与单片机通信采用文本格式：`T25H60S45L80`

| 字段 | 含义 | 示例 |
|------|------|------|
| T | 温度(°C) | T25 |
| H | 湿度(%) | H60 |
| S | 土壤湿度(%) | S45 |
| L | 光照强度 | L80 |

## 指令格式

| 指令 | 说明 |
|------|------|
| `STATUS\r\n` | 请求立即发送传感器数据 |
| `WATER\r\n` | 立即浇水 |

## 蓝牙配置

- 模块：HC-08 (BLE 蓝牙 4.0)
- 波特率：9600

## 构建 APK

1. 在 HBuilderX 中打开项目
2. 菜单 → 发行 → 原生App-云打包 → Android
3. 勾选"使用公共测试证书" → 打包

## 使用说明

1. 安装 APK 到 Android 手机
2. 开启蓝牙和定位权限
3. 点击"扫描"搜索 HC-08，或点击"手动"输入 MAC 地址
4. 连接成功后自动显示传感器数据
