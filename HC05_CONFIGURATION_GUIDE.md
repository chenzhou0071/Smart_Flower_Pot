# HC-05蓝牙模块配置指南

## 🎯 重要信息

### AT模式操作
1. **进入AT模式**：长按HC-05模块上的按键，然后上电，直到红蓝灯变为慢闪
2. **AT模式波特率固定**：38400，不可更改
3. **退出AT模式**：重新上电HC-05模块（或断电重启）

### 波特率说明
- **AT指令波特率**：始终为38400
- **通信波特率**：可以通过AT指令设置（如115200）
- **重要**：设置新的通信波特率后，需要重新初始化USART

### 密码限制
- **密码长度**：只能设置4位数字
- **默认密码**：1234或0000
- **主从配对**：相同密码的主从模块可以相互通信


### 软件配置
- **初始波特率**：38400（HC-05默认）
- **通信波特率**：38400（使用默认，不进行AT配置）
- **数据位**：8位
- **停止位**：1��
- **校验位**：无

## 🔧 配置步骤

### 方案一：使用默认配置（推荐）
```c
// 直接使用HC-05默认配置，无需AT模式
HC05_Init();  // 使用38400波特率

// 手机APP设置波特率为38400
```

### 方案二：自定义配置
1. **手动进入AT模式**
   - 长按HC-05按键，然后上电
   - 等待红蓝灯慢闪

2. **调用AT配置函数**
   ```c
   HC05_AT_Init();  // 发送AT指令配置
   ```

3. **重新上电HC-05**
   - 断电重启HC-05模块
   - 进入正常工作模式

4. **更新手机APP波特率**
   - 如果设置了115200，手机APP也要设为115200

## 📊 AT指令示例

### 基本指令
```
AT                    // 测试AT模式
AT+NAME=Irrigation    // 设置设备名称
AT+BAUD=8             // 设置波特率115200
AT+ROLE=0             // 设置为从机
AT+PSWD=1234          // 设置密码1234
AT+CMODE=0            // 固定连接模式
```

## 🚨 常见问题

### 1. 数据乱码
- **原因**：波特率不匹配
- **解决**：确认HC-05和手机APP使用相同波特率

### 2. AT指令无响应
- **原因**：
  - 未正确进入AT模式
  - 波特率不是38400
  - 接线错误
- **解决**：
  - 确认红蓝灯慢闪
  - 使用38400波特率
  - 检查TX/RX接线

### 3. 无法连接蓝牙
- **原因**：
  - HC-05未上电
  - 密码错误
  - 距离太远
- **解决**：
  - 检查3.3V供电
  - 确认密码1234
  - 靠近手机(<10cm)

### 4. 指令无反应
- **原因**：手机APP未设置换行符
- **解决**：勾选"发送新行"或"\r\n"
