# HC-05蓝牙模块接线指南（USART1版本）

## 🔌 正确接线方式

### HC-05模块 → STM32F103C8T6

```
HC-05模块          STM32F103C8T6
─────────────────────────────
TX        →       PA10 (USART1_RX) ⭐
RX        →       PA9 (USART1_TX)  ⭐
KEY       →       PA4 (控制引脚)
VCC       →       3.3V ⚠️⚠️⚠️
GND       →       GND
```

## ⭐ 重要变化

**现在使用USART1 (PA9/PA10)**，而不是USART2：
- **PA9**: USART1_TX - 发送数据到HC-05
- **PA10**: USART1_RX - 接收HC-05数据

## ⚠️ 重要注意事项

1. **电压警告**: HC-05模块必须使用 **3.3V** 供电，使用5V会损坏模块！
2. **交叉连接**:
   - HC-05的TX → STM32的RX (PA10)
   - HC-05的RX → STM32的TX (PA9)
3. **KEY引脚**: 正常工作时PA4输出低电平，AT配置时输出高电平
4. **串口选择**: 使用USART1，不是USART2

## 📍 引脚位置参考

### 在你的STM32开发板上：
- **PA9**: USART1_TX (通常在A排第9个引脚)
- **PA10**: USART1_RX (通常在A排第10个引脚)
- **PA4**: 普通IO口 (通常在A排第4个引脚)

### HC-05模块引脚：
```
┌─────────┐
│  EN/KEY │ ← 接PA4
│  VCC    │ ← 接3.3V
│  GND    │ ← 接GND
│  TX     │ ← 接PA10 (RX)
│  RX     │ ← 接PA9 (TX)
└─────────┘
```

## 🔧 接线步骤

1. **断电状态下接线**（确保STM32和HC-05都断电）
2. **先接电源**: VCC→3.3V, GND→GND
3. **再接数据**: TX→PA10, RX→PA9
4. **最后接控制**: KEY→PA4
5. **检查接线**: 确认没有短路和接反
6. **上电测试**: 先给STM32上电，系统会自动初始化HC-05

## 📱 测试连接

1. **上电后HC-05的LED会闪烁**（表示等待配对）
2. **手机搜索蓝牙设备**: "IrrigationSystem"
3. **输入配对密码**: 1234
4. **HC-05的LED变为常亮**（表示已连接）

## 🎯 技术参数

- **设备名称**: IrrigationSystem
- **配对密码**: 1234
- **使用串口**: USART1
- **波特率**: 115200 (透传), 38400 (AT)
- **通信距离**: ≤10米

## 📊 蓝牙指令

```
STATUS          # 查看系统状态
WATER_NOW       # 立即浇水
```

## 📱 数据格式

每5秒自动上报（仅在连接时）：
```
DATA: T=25.3C,H=65.2%,S=45%,L=78%,W=500
```
