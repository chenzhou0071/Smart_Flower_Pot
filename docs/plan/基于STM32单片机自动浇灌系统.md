| 核心控制器 | STM32F103C8T6最小系统板 |
| 传感器模块 | 热敏电阻传感器 |
|     | 光敏电阻传感器 |
|     | 土壤湿度传感器 |

**二、模块对应通信/连接方式**

|     |     |     |
| --- | --- | --- |
| 模块名称 | 接口类型 | 通信/连接方式 |
| 热敏电阻/光敏电阻/土壤湿度传感器 | ADC模拟输入接口 | 模拟信号采集，间歇供电 |
| 微型抽水泵（含驱动模块） | GPIO数字输出接口 | 高低电平控制通断 |
| OLED显示屏 | I2C/SPI接口 | 串行通信 |
| HC-05蓝牙模块 | USART串口接口 | 串口通信+蓝牙无线传输 |
| 太阳能电源管理模块 | 电源接口（VCC、GND） | 稳压供电连接 |

**三、独立开发单元及目标**

传感器模块：完成温度、光照、土壤湿度数据采集，实现稳定获取环境参数的功能。

微型抽水泵+驱动模块：实现水泵的启停控制，达成精准滴灌出水的效果。

OLED显示模块：实现字符、图形（笑脸/哭脸）显示，完成环境参数与植物健康状态的可视化呈现。

HC-05蓝牙模块：实现与核心控制器的串口通信，达成数据无线传输的功能。

太阳能电源管理模块：实现太阳能转换与稳压输出，为整个系统提供稳定供电。

手机App：实现蓝牙搜索配对、数据接收解析与显示，达成参数超限报警的功能。
