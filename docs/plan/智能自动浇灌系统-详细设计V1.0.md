# 智能自动浇灌系统 - 详细设计文档

**版本**：V1.0
**日期**：2026-03-12
**项目**：基于STM32单片机自动浇灌系统

---

## 一、系统概述

### 1.1 设计目标

实现一个能够自动监测土壤湿度、温度、光照环境参数，并根据湿度阈值自动灌溉的智能系统。同时支持蓝牙无线传输数据至手机App进行查看和控制。

### 1.2 系统架构

```
┌─────────────────────────────────────────────────────────────────┐
│                         手机App                                  │
│                     (蓝牙连接 / 数据展示)                         │
└─────────────────────────┬───────────────────────────────────────┘
                          │ 蓝牙无线传输 (HC-05)
┌─────────────────────────▼───────────────────────────────────────┐
│                    STM32F103C8T6 主控板                         │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐  ┌──────────┐        │
│  │ 传感器   │  │  OLED   │  │ 蓝牙模块  │  │ 水泵驱动  │        │
│  │ 模块     │  │ 显示屏   │  │  HC-05   │  │  模块     │        │
│  └──────────┘  └──────────┘  └──────────┘  └──────────┘        │
└─────────────────────────┬───────────────────────────────────────┘
                          │

│                  5V/1A + 3.3V/1A 输出                            │
└─────────────────────────────────────────────────────────────────┘
```

## 二、硬件设计

### 2.1 核心控制器

- **型号**：STM32F103C8T6 最小系统板
- **工作电压**：3.3V
- **资源使用**：
  - ADC通道：3个（土壤湿度、温度、光照）
  - GPIO：若干（OLED、水泵控制、指示灯）
  - USART：1个（蓝牙通信）
  - I2C：1个（OLED显示）

### 2.2 传感器模块

| 传感器 | 功能 | 接口 | 供电 |
|--------|------|------|------|
| 土壤湿度传感器 | 检测土壤含水量 | ADC | 间歇供电 |
| NTC热敏电阻 | 检测环境温度 | ADC | 3.3V |
| 光敏电阻(GL5528) | 检测环境光照 | ADC | 3.3V |

**设计说明**：
- 土壤湿度传感器采用间歇供电设计，每隔一段时间通电测量，测量完毕后断电，可有效延长电极使用寿命
- 传感器信号通过分压电路接入STM32 ADC

### 2.3 执行机构

| 组件 | 功能 | 控制方式 |
|------|------|----------|
| 微型水泵 | 抽水灌溉 | GPIO + MOSFET驱动 |
| 排水电磁阀(可选) | 控制水流 | GPIO |

**水泵控制逻辑**：
- STM32 IO输出高电平 → MOSFET导通 → 水泵工作
- STM32 IO输出低电平 → MOSFET截止 → 水泵停止

### 2.4 显示模块

- **屏幕**：0.96寸 128×64 OLED显示屏
- **驱动芯片**：SSD1306
- **通信接口**：I2C
- **显示：
  - 内容**土壤湿度百分比
  - 环境温度值
  - 光照强度等级
  - 水泵工作状态（运行/停止）
  - 植物状态图标（笑脸/哭脸）

### 2.5 蓝牙通信模块

- **型号**：HC-05
- **通信接口**：USART串口
- **传输方式**：蓝牙无线透传
- **功能**：
  - 向App发送实时数据
  - 接收App下发的控制指令

### 2.6 电源模块

- **输入**：太阳能板或USB供电
- **输出**：
  - 5V/1A → 供 水泵
  - 3.3V/1A → 供 STM32、传感器、OLED、蓝牙模块

## 四、控制算法

### 4.1 灌溉策略：湿度阈值控制

```
土壤湿度 > 阈值上限(例如60%)  →  不浇水
土壤湿度 < 阈值下限(例如40%)  →  启动水泵浇水
土壤湿度 介于 40%~60% 之间    →  保持当前状态
```

### 4.2 防抖动设计

为防止频繁启停，设置以下保护：
- **启动延迟**：湿度低于下限后，需持续30秒才启动水泵
- **停止延迟**：湿度达到上限后，需持续30秒才停止水泵
- **最小运行时长**：水泵每次启动至少运行10秒

### 4.3 传感器数据处理

- **采样方式**：连续采样5次，去掉最大值和最小值，取平均
- **刷新频率**：每秒采样一次，显示屏每秒刷新一次

### 4.4 阈值参数（可调整）

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 湿度阈值上限 | 60% | 停止浇水阈值 |
| 湿度阈值下限 | 40% | 启动浇水阈值 |
| 启动延迟 | 30秒 | 防止误触发 |
| 停止延迟 | 30秒 | 防止误停止 |
| 最小运行时间 | 10秒 | 保证灌溉效果 |

## 五、通信协议

### 5.1 蓝牙数据传输格式

**STM32 → App 发送格式**：
```
[帧头] [数据类型] [数据值] [校验] [帧尾]
 0xAA    0x01      2字节    1字节   0x55
```

**数据类型定义**：
| 类型码 | 含义 |
|--------|------|
| 0x01 | 土壤湿度 |
| 0x02 | 温度 |
| 0x03 | 光照 |
| 0x04 | 水泵状态 |

**App → STM32 发送格式**：
```
[帧头] [指令类型] [参数] [校验] [帧尾]
 0xAA    0x10      1字节   1字节   0x55
```

**指令类型定义**：
| 指令码 | 含义 |
|--------|------|
| 0x10 | 设置湿度阈值下限 |
| 0x11 | 设置湿度阈值上限 |
| 0x20 | 手动开启水泵 |
| 0x21 | 手动关闭水泵 |
| 0x30 | 查询当前状态 |

### 5.2 通信示例

**App发送查询指令**：
```
AA 30 00 BF 55
```

**STM32回复**：
```
AA 01 2D 00 55   (湿度45%)
AA 02 18 00 55   (温度24°C)
AA 03 64 00 55   (光照100%)
AA 04 01 00 55   (水泵开启)
```

## 八、安全与保护

### 8.1 软件保护

| 保护项 | 说明 |
|--------|------|
| 水泵超时保护 | 连续运行超过30分钟自动停止 |
| 蓝牙断连处理 | 断连后设备按默认阈值自动运行 |
| 传感器异常 | ADC值异常时停止灌溉 |

### 8.2 硬件保护

| 保护项 | 说明 |
|--------|------|
| 电源防反接 | 二极管保护 |
| MOS管防护 | 添加续流二极管 |
| 静电防护 | TVS管保护接口 |
